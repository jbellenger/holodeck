# Holodeck Player

A browser-based player for Blender animation presentations generated by the Holodeck Blender add-on.

## Quick Start

1. Place your `manifest.json` and `render/` folder (containing PNG frames) in this directory
2. Make sure `index.html`, `player.js`, and `styles.css` are present alongside them
3. Start the server:
   ```bash
   python3 server.py
   ```
4. Open http://localhost:8000 in your browser

## Keyboard Controls

| Key | Action |
|-----|--------|
| **Space** | Play/pause (auto-stops at markers) |
| **Arrow Left** | Jump to previous marker |
| **Arrow Right** | Jump to next marker |

## File Structure

The player expects:
- `index.html`, `player.js`, `styles.css` - Browser assets
- `manifest.json` - Generated by Holodeck, contains frame paths, fps, and marker positions
- `render/` - Directory containing PNG frames

## Example

The canonical sample Blender project in this repo is `../demo.blend`.
It is the versioned source file for local development and testing with the Holodeck add-on.

## Creating Presentations

1. Install the Holodeck add-on in Blender
2. Create your animation in Blender
3. Add timeline markers at slide transition points
4. Render the animation or run the headless export workflow
5. Copy the exported bundle contents to this directory if you want to use `server.py`
6. Run the server and present!
